# Introducción al Big Data

## 1. ¿Qué es Big Data?
No es solo "muchos datos". Es el tratamiento y análisis de enormes repositorios de datos que son tan desproporcionadamente grandes que resulta imposible tratarlos con las herramientas de bases de datos y analíticas convencionales.

!!! quote "Regla de Oro"
    Si tus datos caben en una hoja de Excel o en una base de datos SQL tradicional y el tiempo de respuesta es aceptable, **NO necesitas Big Data**.

## 2. Las 5 V del Big Data
Para que un problema se considere Big Data, suele cumplir estas características:

* **Volumen:** La cantidad de datos (Terabytes, Petabytes...).
* **Velocidad:** La rapidez con la que se generan y procesan los datos (ej. sensores, streaming).
* **Variedad:** Distintos formatos. No solo texto, también video, audio, logs, redes sociales.
* **Veracidad:** La calidad y fiabilidad del dato (limpiar el "ruido").
* **Valor:** La capacidad de convertir esos datos en dinero o decisiones útiles.

## 3. Tipos de Datos
| Tipo | Descripción | Ejemplo |
| :--- | :--- | :--- |
| **Estructurados** | Tienen formato fijo y definido. | Tablas SQL, Excel, CSV. |
| **No Estructurados** | No tienen estructura interna clara. | Vídeo, Audio, PDF, Emails. |
| **Semi-estructurados** | Tienen etiquetas pero no esquema rígido. | HTML, XML, JSON. |
